## 📱 Cómo Usar

### Para Clientes:
1. Navega por el catálogo de productos
2. Haz clic en un producto para ver detalles
3. Selecciona "Solicitar por WhatsApp"
4. Se abrirá tu chat de WhatsApp con un mensaje pre-escrito
5. Proporciona tu dirección en Cuba
6. Confirma el método de envío (Barco o Avión)
7. Realiza el pago a través de nuestro sistema seguro
8. Recibe tu pedido en Cuba en 5-15 días

### Para Administradores:
- Edita `script.js` para agregar/editar productos
- Modifica `styles.css` para cambiar colores o diseño
- Actualiza contacto de WhatsApp en `script.js`
